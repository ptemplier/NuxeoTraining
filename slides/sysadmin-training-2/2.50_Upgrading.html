Nuxeo Platform - Upgrading

### Context
Maintaining an application also means upgrading versions.
[ADMINDOC/Upgrading+your+Nuxeo+Version](http://doc.nuxeo.com/x/rgc7)
###Upgrade distribution #### You can upgrade from - A LTS version to the next LTS - A Fast Track version to the next Fast Track - A Fast Track version to the next LTS if this is the latest Fast Track before the LTS ####Before any upgrade - Read the Upgrade Notes - Backup database - Backup the data directory - Backup configuration and custom templates - Backup log directory ####Upgrade - Deploy new Nuxeo - Adapt your configuration - Start Nuxeo
