### Context
Now we know what the Nuxeo Platform is about. So what will we need to have in terms of architecture in order to install it properly ?
### Deployment Concepts #### Possibilities - Many specific architectures are possible - Dedicated distribution including only the required bundles - Dedicated nodes - Multi repositories - Client / Server mode... #### What we recommend for most installations - Clustering with: - A load balancer with sticky sessions - A reverse proxy - Nuxeo nodes - A shared database, elasticsearch cluster and filesystem
### Deployment Concepts #### What Nuxeo handles - Clustering - Multiple deployment options #### What you may need to set up - A reverse proxy: Nuxeo should never be exposed directly - A load balancer with sticky sessions
