Choose your version

Fast Track releases
  • Are released every 2 months, between LTS releases
  • Have uneven numbers: 7.x, 9.x...
  • Contain the latest functionalities
  • Are not maintained, and only supported for 4 months => You need to migrate often
  • Are fitted for testing and development, not for production use
LTS releases
  • Are released once a year, around november
  • Have even numbers: 6.0, 8.0...
  • Are maintained for 3 years
  • Are fitted for production use
### Practice
First, get the application and start it.
#### Download and Start the Nuxeo Server - Get the latest LTS release, as a multi platform zip file - Unzip it in the home directory - Start it using a terminal: /path/to/server/bin/nuxeoctl start - Go to http://localhost:8080
### Configuring the Nuxeo Platform #### Setup Wizard at first start ![wizard](img/nxp-install/wizard.png) - Use it for common configuration (database, proxy, smtp, modules to install...)
### Configuring the Nuxeo Platform #### Distributing The Application - ZIP distribution is pre-configured with default paths - VM distribution is pre-configured with recommended paths - Other installers marry common usages and Nuxeo recommendations ##### Server (`NUXEO_HOME` environment variable) - `/home/nuxeo/nuxeo-dm/` or `/opt/nuxeo/` ##### Configuration (`NUXEO_CONF` environment variable) - `$NUXEO_HOME/bin/nuxeo.conf` (default) - `/etc/nuxeo/nuxeo.conf` (recommended) ##### Data (`nuxeo.data.dir`) - `$NUXEO_HOME/nxserver/data` (default) - `/var/lib/nuxeo` (recommended) ##### Logs (`nuxeo.log.dir`) - `$NUXEO_HOME/log/` (default) - `/var/log/nuxeo` (recommended)
