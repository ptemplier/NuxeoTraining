###Avant de démarrer - Quelques règles d'usage #### Attachez votre ceinture et... - N'hésitez surtout pas à poser des questions, quand bon vous semble - Eteignez votre téléphone ou passez le en mode silencieux - Suivez bien les instructions données pour que tout le monde suive au même rythme - Vous ne pourrez pas voir toutes les fonctionnalités disponibles, mais : - S'il vous semble que quelque chose d'important pour vous a été laissé de côté ou ne sera pas abordé, prévenez moi ! - Nous allons vous montrer les éléments progressivement. ##Installation et démarrage
###Téléchargement Disponible depuis nuxeo.com : http://www.nuxeo.com/fr
 Packages disponibles: - Package multi-plateforme (.zip) : Serveur Nuxeo complet zippé - Installeur Windows (.exe) : installe les dépendances si nécessaire (Java, LibreOffice, etc.) - Installeur Debian/Linux : installe les dépendances et un script init - Images de machine virtuelles
### Installation
Toutes les procédures d'installation : doc.nuxeo.com
#### Prérequis Java 7 ou Java 8 #### Emplacement - Où : Racine d'un disque (C:\ par exemple) ou /home/utilisateur sous linux - Pas de problème de profondeur de chemin - Pas de problème de chemin avec des espaces / caractères spéciaux
###Démarrage - Sous Windows : Double-clic sur Start Nuxeo.bat - Sous Mac : Double-clic sur Start Nuxeo.command - Sous linux : cd chemin/du/serveur/nuxeo/bin puis ./nuxeoctl start - Panel Control pour arrêter ou relancer le serveur ou ./nuxeoctl stop ./nuxeoctl restart sous linux - Accéder à Nuxeo : http://localhost:8080/nuxeo
### Exercice - Installer la plateforme Nuxeo - Dézippez la plateforme Nuxeo dans un espace approprié - Lancez la plateforme Nuxeo - Une fois la plateforme démarrée, ouvrez l'application dans le navigateur
### Exercice - Assistant de configuration #### Que voyez-vous ? Au premier démarrage du serveur : un assistant - Pour vous aider à régler les paramètres de configuration les plus courants. #### Renseignez les paramètres de configuration - **Base de données :** h2 (base de données embarquée)
Tests, démo, développement uniquement - **SMTP :** pour l'envoi de notifications par email aux utilisateurs
**Utiliser le port 25000 pour la formation** - **Proxy** : aucun sauf si votre entreprise utilise un proxy pour accéder à internet - **Connect :** Enregistrez vous pour accéder à la Marketplace et à Nuxeo Studio - **Modules :** Cocher DAM, Nuxeo Drive, Nuxeo Diff, Nuxeo Spreadsheet #### Redémarrez la plateforme
