##Workflows

## Nuxeo Quota
###Contexte

La gestion des quotas est un besoin récurrent.
Alors voyons comment les gérer facilement.

###Nuxeo Quota ####Principes - Addon disponible directement depuis la Marketplace Nuxeo - Vous permet de définir un quota sur les domaines et les espaces de travail - Les quotas sont hérités de leur parent - On peut définir des quotas sur les utilisateurs personnels
###Nuxeo Quota ####Utilisation

1 - Installer l'addon

2 - Initialiser les statistiques sur les documents existants

3 - Définir des quotas

  • Les quotas peuvent être définis depuis :
    • l'Admin Center (pour les espaces personnels)
    • chaque document folderish

Droits nécessaires

  • Les informations sur les quotas sont visibles par n'importe quel utilisateur avec le droit de lecture sur un espace
  • Il faut le droit "Gérer tout" sur un espace pour pouvoir définir ses quotas
###Exercice Quotas #### Définir un quota sur les espaces personnels - Installez l'addon Nuxeo Quota depuis l'onglet Admin Center > Update Center > Nuxeo Marketplace Packages - Initialisez les statistiques sur votre instance - Définissez un quota sur les espaces personnels

## Nuxeo Drive
###Contexte

Ce serait tellement plus facile si je pouvais automatiquement synchroniser mon travail sur mon instance... Hé, mais peut-être que Nuxeo a quelque chose pour ça ?

###Objectif - Synchroniser des dossiers / espaces de travail avec son poste de travail
###Nuxeo Drive #### Principes - Synchronise les espaces Nuxeo avec votre ordinateur - Une fois qu'un dossier est synchronisé, vous pouvez modifier les documents directement depuis Nuxeo #### Installation - Côté serveur : addon disponible directement depuis la Marketplace Nuxeo - Côté client : installation d'un logiciel
###Nuxeo Drive #### Utilisation - Après l'installation de l'addon, allez sur Home > Nuxeo Drive pour télécharger le logiciel client - Installez le logiciel et donner les informations sur votre serveur. Par ex :
http://localhost:8080/nuxeo
Administrator / Administrator - Utilisez l'icône de synchronisation pour lancer une synchro - Créez / Modifiez vos documents directement depuis votre ordinateur ou utiliser l'icône de modification en ligne pour les ouvrir
###Exercice Drive #### Synchroniser son espace personnel - Installez l'addon Nuxeo Drive depuis l'onglet Admin Center > Update Center > Nuxeo Marketplace Packages - Synchronisez votre espace personnel avec votre poste de travail - Créez un fichier localement et voyez s'il est placé sur le serveur

###Contexte

La librairie "Les meilleurs livres" a décidé d'utiliser Nuxeo pour gérer son catalogue de livres.

Et c'est vous qui allez être chargé de mettre en place leur projet.

Les spécifications sont en cours d'écriture. En attendant, il faut vous familiariser avec les outils que vous allez utiliser.

##Découvrir Nuxeo Studio
###Nuxeo studio ####Nuxeo Studio est : - un outil graphique pour configurer des applications basées sur la Plateforme Nuxeo - un service en ligne (SaaS) - un canal de livraison et de déploiement via l'Update Center #### Avec Nuxeo Studio, vous pouvez : - Configurer et adapter votre application - qu'il s'agisse de vos types de documents ou de la mise en place de règles métier - sans aucun développement - Economiser du temps et de l'argent, en vous concentrant sur vos besoins métier - Vous assurer d'avoir une maintenance à long terme de votre application grâce à l'option de déploiement "target platform" (version cible)
###Nuxeo Studio - Comment ça marche - Accès direct via le portail Connect - 30 jour d'essai gratuit - Fournit un JAR qui contient toute la configuration de votre projet - Studio ne produit aucun code, uniquement du XML. Votre instance Nuxeo se connecte à Studio et déploie la configuration. - Intégration d'outils de développement (Maven et Nuxeo IDE)
###Nuxeo Studio - Navigation et fonctionnalités #### Informations sur votre projet Vous avez les informations les plus importantes sous les yeux, tout de suite.
###Nuxeo Studio - Navigation et fonctionnalités #### Menu Project Configurez votre application en ajoutant des nouveaux composants.
##### Boutons - New : Ajoutez un nouvel élément - Import : importez des modèles d'application pré-définis - Download : Téléchargez votre projet dans un fichier JAR - Commit : commitez vos modifications dans l'outil de gestion des versions de Studio ##### Sous-menus - Resources : uploadez de nouveaux éléments (images, fichiers de traduction...) - Branding : définissez votre logo, vos couleurs... - Content model : créez vos types de document, vos schémas (métadonnées)... - Workflows - Listing and views : personnalisez la navigation et la présentation de vos documents - Automation : implémentez votre logique métier - Templates : création de modèles d'email et de documents - Roles and permissions : définissez vos propres permissions et les utilisateurs / groupes par défaut - Vocabularies : gérez les listes d'options pré-définies - Advanced settings : allez encore plus loin dans la configuration
###Nuxeo Studio - Navigation et fonctionnalités #### Menu Settings and versioning Définissez votre version cible et les options générales de votre projet.
- Project settings : définissez les réglages généraux de votre projet, dont la version cible - Versioning : gérez les versions de votre projet grâce à l'outil de gestion des versions intégré à Studio - Application templates : importez des modèles d'applications pré-définies (même chose que le bouton Import) - Registries : fonctionnalité avancée - déclarez des éléments de code personnalisé à utiliser dans Studio
###Exercice Nuxeo Studio (1/2) #### Découverte de Nuxeo Studio * Connectez-vous à votre compte Nuxeo Studio, soit par l'URL directe, soit depuis Nuxeo DM
`Admin center` > `Update Center` > `Studio` > lien `Go to Nuxeo studio` * Dés-importez le modèle par défaut DM
`Settings and versioning` > `Application templates` > `Nuxeo DM default configuration` > `Remove` * Créez un nouveau branding appelé `bestBooksTheme`
`Project` > `Branding` * laissez les valeurs par défaut, à l'exception du logo: files/bestBooksLogo.png * Enregistrez
###Exercice Nuxeo Studio (2/2) #### Testez vos modifications * **Déployez votre projet Studio sur votre instance Nuxeo DM**
`Admin Center` > `Update Center` > `Nuxeo studio` > `Update` * Activez le mode développement pour bénéficier du hot reload * Redémarrez votre instance * Vérifiez que votre projet est correctement déployé (logo)
