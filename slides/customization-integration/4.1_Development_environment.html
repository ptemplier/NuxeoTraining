# Nuxeo Training - Development environment and project structure

Software Required

  • Java 7
  • Eclipse 3.6 or 4.2 and Nuxeo IDE from the eclipse marketplace
  • Nuxeo server with SDK
### Nuxeo SDK server #### Concepts - A complete Nuxeo Distribution and even more:  - Don't have to deal with dependencies - Test bundles - Sources - Bundle hot reload configuration - Mapping for maven synchronization
### Nuxeo IDE #### Concepts - Tools with template examples to ease Nuxeo development - Connects to your studio projects to retrieve its configuration - Integrated unit test system - Easy deployment of your development on your local server - Allows to launch the server in debug mode http://doc.nuxeo.com/x/aoKE
### Practice - Nuxeo IDE #### Link your Nuxeo IDE with your connect account - Install eclipse and Nuxeo IDE - Unzip your Nuxeo server SDK in an easy to remember place on your local disk - Go to the eclipse preferences menu - In the Nuxeo submenu: - Choose `Nuxeo connect` and fill your connect account information - Choose `Nuxeo SDK` and select your Nuxeo server SDK folder
### Practice - Nuxeo IDE #### Create your first Nuxeo plugin project Now that your connect account is linked, let's start a project! - On top of your screen search for the Nuxeo wizard's NX icon - Choose Nuxeo plugin project - Fill in the necessary information: - Project id: `best-books` - Root package: `org.nuxeo.training.bestbooks` - Click on `next`, then `finish`

Project structure

  • src/main/java: Java files and packages
  • src/main/resources: Resource files
    • META-INF/MANIFEST.MF: Bundle ID card
    • OSGI-INF/extensions: Lists components to deploy, and allows to express dependencies between them
    • OSGI-INF/deployment-fragment.xml: Contains bundle installation instructions, including bundle's dependencies
  • src/test/java: Junit tests
### Practice - Nuxeo IDE #### Link your plugin project with your studio project - Right click on your project name and choose `properties` - In the `Nuxeo` submenu, check your studio project and press OK

The Nuxeo IDE perspective

Top left tabs

Project explorer tab

Used to navigate into your project structure.

Nuxeo components tab

Used to browse the different components provided (similar to explorer.nuxeo.org)

Nuxeo studio tab
  • Shows your studio project elements
  • Allows you to refresh them in case of a modification
  • Allows you to export components created in Nuxeo IDE into your studio project's registries

The Nuxeo IDE perspective

The Nuxeo server central tab

Provides buttons to interact with your local nuxeo server:

Deployment configuration

To choose which projects should be deployed on the server.

Start the server

Note that Nuxeo will use the nuxeo-sdk.conf file for its configuration.

Start the server in debug mode

Allows to suspend the server execution when stumbling on a break point.

Stop the server

Reload project

Export your code directly on the server for testing.
