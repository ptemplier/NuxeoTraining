# Nuxeo training - Registries
### Context

You have received one more request from the Best Books library's staff.

They would like to isolate their specific document types in the documents type list when creating a new document.

###What we want to do ####At the end of the course: - Create a BestBooks document category to visually differentiate them upon creation
### Registries #### What for? - Make available missing docTypes, layouts, etc… in Studio - The existing ones in Nuxeo but missing in Studio registries  - The custom ones done in another project  - Add new elements that are not configurable yet in Studio #### How to use them - Found in Nuxeo studio's `Settings and versioning` > `Registries` menu - Use JSON format to declare your elements - Examples are provided to get you started (Click on "see example", then on the "Insert" button)
### Practice - Registries - Create a `BestBooks` document category - Apply your document category to the `Book` and `BookLibrary` document types
