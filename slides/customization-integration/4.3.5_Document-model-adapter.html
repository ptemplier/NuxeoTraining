# Nuxeo Training - Document Model Adapter
### Document Model Adapter #### Concepts - Wrapper for your document types - Wraps document types attributes and possibly more logic (lifecycle transitions, etcâ€¦) #### Advantages - Simplifies code writing, ex: `bookAdapter.getPublicationDate()`
(no need to know about the schemas used, allows autocomplete) - Refactoring friendly: document model is not accessed directly - Fully integrated with Nuxeo IDE
### Document Model Adapter #### Creating a document model adapter - Use the NX icon to launch the Nuxeo wizard and select the `Document Adapter` option - Select from which schemas you want to create your adapter, and you're done! - You may possibly add more elements (lifecycle transitions, business rules methods) for even easier use. #### Calling a document model adapter 
MyDocTypeAdapter myDocTypeAdapter = documentModelVariable.getAdapter(MyDocTypeAdapterClass.class);
### Practice - Document model adapter #### Creation - Create a document model adapter for the `book` schema.
You can name it `BookAdapter` - Add the `to_inLibrary` and `to_borrowed` transitions to your adapter #### Usage - Create a `TestBookAdapter` class and a `testBookAdapter()` method that will: - Create a `Book` at the documents root - Set an ISBN value for the Book - Set the Book as borrowed - Saves the document into the repository - Assert the operations you just made - Run your test and check the result Use your adapter whenever possible of course :)
