# Nuxeo training - Using elasticsearch in Content Views
### Context

This is time to add a nice book search form to our application. Leveraging elasticsearch features could be a great idea!

###What we want to do ####At the end of the course: - A custom search form leveraging elasticsearch features
### When Is elasticsearch useful ? #### elasticsearch Key Points - Better performances - But accurate data is not guaranteed immediately #### Use elasticsearch For - Content views not requiring immediate exact results - Costly queries - Fulltext searches - Advanced search features #### Not Always Recommended For - Document content listings - Any query requiring guaranteed up to date information
### Custom Search Forms #### Where To Create Custom Search Forms - Default search form: Listing and views > Search - Alternate search forms: Listing and views > Content views with Search content view flag
### Leveraging elasticsearch
elasticsearch aggregates available since 6.0
#### elasticsearch Specific Widgets: Aggregates ##### Histogram Widgets - Ranges are calculated automatically (date or number) ##### Range Widgets - You define ranges by yourself (date or number) ##### Terms - Frequent words are proposed automatically - Metadata can be mapped to a vocabulary, users, documents
### Using Range Widgets #### How To Use Range Widgets - Format is mandatory (ex: dd-MM-yyyy) - From and to fields - Need an expression - But can be left empty to mean "anywhere in the past (or future)" #### Date Ranges Examples - Past: From (empty) to now - Future: From now to (empty) - Last week: From now-7d to now - Yesterday: From now-24H to now
### Practice #### Leveraging elasticsearch
Create an alternate search form to search for books
##### What To Search For - Their publication date using a date range widget, with the following ranges: - Last month - Last week - Future - Their category (using a terms widget) ##### Results display - Checkboxes should be shown on screen - The results should use the `bookTableLayout` - Add a title to your search form using the `advanced configuration` zone in the `results` tab
