# Nuxeo training - XML extensions
### Context

Users are facing a challenge: they want to give people write access without letting them remove documents.

### What we want to do #### At the end of the course - Create a new "ReadWriteWithoutRemove" permission and show it into the UI.
### XML extensions #### Concepts - Nuxeo exposes extension points almost everywhere - These extension points can be configured using XML (what studio does) - Not all extension points are configurable in studio yet - For these, studio integrates the XML extension functionality
### XML extensions #### How does it work? (1/2)
  • Go to explorer.nuxeo.org
  • Select your platform version
  • Go to the extension points
  • Search for a relevant term
  • Look at the proposed extension points, each has its documentation
  • Look at the existing contributions to see the default behavior
### Practice
Create a new permission "ReadWriteWithoutRemove" and show it into the UI
#### Check for the existing - Look for the permission keyword in the existing extension points - Have a look at what the securityService offers - Check the existing XML contributions built into Nuxeo core #### Build your own permission - Create some XML for a new permission containing the following permissions: - Read - All permissions included into the Write permission except the Remove permission #### Show it into the UI - Create some XML to show your new permission into the UI, just before the Write permission
