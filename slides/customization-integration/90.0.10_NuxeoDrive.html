# Nuxeo Training - Nuxeo Drive Add-On
###Context

How easy it would be if I could just synchronise my work with my repository automatically... wait, maybe Nuxeo has something to offer in that regard ?

###What we want to do ####At the end of the course: - Synchronise folders / workspaces with our computer
###Nuxeo Drive Add-On ### Principles - Synchronizes server spaces with your computer - Once a folder is synchronized, you can edit files directly from the server #### Installation - Server-side, add-on directly available from the Nuxeo marketplace - Client-side, a software needs to be installed
###Nuxeo Drive Add-On #### UI usage - After add-on installation, go to home > Nuxeo drive to get the client software - Install software and link it to your server, ex:
http://localhost:8080/nuxeo
Administrator / Administrator - Use the synchronize icon to launch synchro - Create / edit documents directly from your computer, or use the live edit icon to open them
