# Nuxeo Training - Development environment and project structure

Software Required

  • Java 7
  • Eclipse 3.6 or 4.2 and Nuxeo IDE from the eclipse marketplace
  • Nuxeo SDK

Project Structure

  • Following Maven project structure 
    • pom.xml: the Maven descriptor
    • src/main/java: Java files and packages
    • src/main/resources
      • META-INF/MANIFEST.MF
      • OSGI-INF
      • OSGI-INF/deployment-fragment.xml

Nuxeo IDE

    http://doc.nuxeo.com/x/aoKE
  • Tools with template example to ease Nuxeo development 
  • Connection to studio 
    • Access to document properties registry 
    • Can unit test 
    • Can easily launch Studio + Java plugin in a Nuxeo SDK server in debug mode 
    • Automatically export your custom operation 

Nuxeo SDK server

  • A complete Nuxeo Distribution and even more : 
    • Don't have to deal with dependencies 
    • Test bundles 
    • Sources 
    • Bundle hot reload configuration 
    • Mapping for maven synchronization 

Exercise - Your first bundle

Create a Nuxeo project

  • With Nuxeo IDE, create a new Nuxeo Project. Choose a name, nuxeo-book-management can be a nice choice 
  •  It will automatically 
    • Creates folders
      • src/main/java and src/main/resources 
      • OSGI-INF and META-INF folders in resources 
    • Creates a MANIFEST.MF file in META-INF 
  • Bind the project to your Studio project
###Create a new Automation gadget to retrieve all the Books - Needs `nuxeo-dm` module installed - Use the Nuxeo IDE Automation gadget wizard - Deploy your changes in your SDK - Check that your Automation gadget is available in the user dashboard

Deploy Your Bundle (without the sdk)

  • Synchronize your pom (you may have new dependencies)
  • Build the jar with Maven
    • mvn clean package: package your project as a JAR 
    • mvn clean install: package your project and install it in your local Maven repository 
    • The jar file is in the working directory: target folder
    • Dependency issues ? See Nuxeo Maven Usage documentation
  • Put your JAR archive in the bundles folder on a standard Nuxeo Server (not sdk) 
  • (nxserver/bundles) for those who don’t remember 
  • Check that your Automation gadget is available
