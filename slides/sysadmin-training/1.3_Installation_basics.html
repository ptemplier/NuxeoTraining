###Architecture
  • ZIP distribution is pre-configured with default paths
  • VM distribution is pre-configured with recommended paths
  • Other installers marry common usages and Nuxeo recommendations
- Server (`NUXEO_HOME`) - `/home/nuxeo/nuxeo-dm/` or `/opt/nuxeo/` - Configuration (`NUXEO_CONF`) - `$NUXEO_HOME/bin/nuxeo.conf` (default) - `/etc/nuxeo/nuxeo.conf` (recommended) - Data (`nuxeo.data.dir`) - `$NUXEO_HOME/nxserver/data` (default) - `/var/lib/nuxeo` (recommended) - Logs (`nuxeo.log.dir`) - `$NUXEO_HOME/log/` (default) - `/var/log/nuxeo` (recommended)
