### Nuxeo – The Company - Founded in 2000, more than 500 customers and 5 000 installations - ~50 people in the company, 70% engineers - Based in France, New York and California - Nuxeo supports customers in creating, building, maintaining, deploying and operating applications - Nuxeo Connect subscription offering for maintenance, support, and access to online services - Nuxeo covers the full lifecycle of applications: - Application Designer: Nuxeo Studio - Development Environment: Nuxeo IDE (Eclipse), Maven Tooling - Testing Toolset: Unit, Functional, Integration, Performance tests with Nuxeo Runtime Tests, Selenium, WebDriver, FunkLoad - Deployment tools: Nuxeo Marketplace, Update Center
### Nuxeo – The Platform - Nuxeo Platform is a full stack software platform for building content-centric business applications - Designed for software developers, architects and business managers who create software for internal use or for customers - A foundation for business applications in the areas of content management, document management, digital asset management, and case management - Trusted by large organizations for mission-critical applications #### Standard technologies - Based on Java standards Java, OSGi, JTA, JAAS, JMS, EJB3, JAX-RS, JAX-WS, CMIS, … - Uses many open source frameworks / libraries : JBoss Seam / Hibernate / Apache commons - Can run in any Servlet container, Java EE Application server, bare JVM or OSGI container, typically: Tomcat, JBoss, Jetty #### Easy to customize, extend and plug - No need to rewrite the source code - Offers content oriented APIs - Nuxeo Studio and Content Automation are powerful
### Nuxeo – Trusted by Great Companies
### Nuxeo − It's All About Content - Nuxeo Platform helps you build Content Applications to effectively manage your content, but what does this mean? - Content: Any information or knowledge, structured or semi-structured, that you are dealing with in your business activity. It can be Documents, Product Specifications, Invoices, Media, Cases, Standards, Specifications, Quotes... - Content Applications: The software that you and your users use to manage and collaborate on content. It can be any business application. Document Management, Digital Asset Management and Case Management are obvious examples, but it could also be an Invoicing application, e-Learning, Catalogs, News and Editorial systems...
###Nuxeo − Type of Projects - Intranet - Collaborative Content Management - Business Workflows: Invoice Management, Purchase Requests, Contract Management, Software Documentation Management - Documents Repository: Quality, Process, ... - Asset Repository - Website Content Management - Mail Management
###Nuxeo − Platform Provides - Advanced Content Repository: Nuxeo Core - Smart storage with scalability and clustering capabilities - Content Model (schemas, files, metadata, etc.) - Facets and mixins - Versioning, Access Control, Relationships - Multi-axis browsing: content tree, metadata, tags, etc. - Full-text indexing with complete query system
###Nuxeo − Platform Provides - Extensible and Configurable ECM services - Document storage, workflow, indexing, security, relation management, … - UI building blocks and Frameworks - Web, RIA, Rich client, … - High level frameworks - Content Application Platform, Case Management Framework, … - Ready to use pre-packaged applications and addons - Nuxeo DM, Nuxeo DAM, Nuxeo Correspondance, ...
###Nuxeo − Platform Features Overview - Rendition management - pdf, HTML preview, web-view - pluggable transformers - Annotation management - Process management - life-cycle, workflows, page-flows, listeners - Mail management - Feed management - Collaboration - notifications, locking, tasks, comments - OpenSocial - Activity stream, user networks
###Nuxeo − Platform Features Overview - Publishing, Synchronization - Digital asset management - pictures, tiling, meta-data, music, video, ... - Personal portal / dashboard - OpenSocial container - Web tools - wiki, blogs, websites - Referential management - users, groups, vocabularies - Admin Center - Manage configuration, monitoring, updates - Delegated administration - Security and users, Local configuration
###Nuxeo − Platform Features Overview - And Much More via Marketplace or Standalone Addons - Reporting (BIRT, ...) - Semantic entities extraction - Automatic Categorization, Bookmarks - Bulk Import - Connectors (GSA, ...) - Signature - Authentication (Shibboleth, OAuth, CAS2, ...) - Quota - Diff - Drive - ...
