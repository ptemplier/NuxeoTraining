# Nuxeo training - Web Services Filtering
### Context

Security is a main concern, especially when calling an application remotely. What can we do about it ?

### Web Services Filtering #### Concepts - Almost all operations and chains are open by default to non-admins - To restrict or disable their usage, we need to setup filters - Filters can be created in the Advanced Settings > Web services filtering option in Studio
### Practice #### Restrict the publishMeeting chain - Define a web services filtering - Add the publishMeeting chain and restrict it to the administrators and communication groups
